## Install 

How to [install Aider](https://aider.chat/docs/install.html) on Mac:

```
python3 -m pip install aider-install

python3 -m pip show aider-install

~/Library/Python/3.9/bin/aider-install

aider --model gpt-4o --openai-api-key
```
